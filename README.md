# Утилита фильтрации содержимого файлов

Консольная программа на Java, которая читает входные файлы, содержащие вперемешку:

- целые числа
- вещественные числа
- строки

и записывает их в разные выходные файлы.

---

## Требования

- Java: 17 или выше
- Система сборки: не используется (компиляция через javac)
- Сторонние библиотеки: отсутствуют (используется только стандартная библиотека Java)

---

## Компиляция и запуск

### 1. Скомпилировать программу

```bash
javac FileFilterUtility.java
```

### 2. Запустить программу

```bash
java FileFilterUtility [опции] <входные_файлы>
```

### пример запуска

```bash
java FileFilterUtility -s -a -p sample- in1.txt un2.txt
```

---

# выходные файлы

по умолчанию создаются в текущей папке:

- sample-float.txt
- sample-integers.txt
- sample-strings.txt

# Статистика

## Краткая (```-s```)

Выводятся только количество элементов каждого типа

## Полная (```-f```)

для чисел:

- количество
- минимальное значение
- максимальное значение
- сумма
- среднее

для строк:

- количество
- длина самой короткой строки
- длина самой длинной строки

---

# Особенности реализации

- Чтение файлов происходит построчно
- Тип данных определяется автоматически:
    - ```long``` -> целое число
    - ```Double``` -> вещественное число
    - иначе -> строка
- Вещественными считаются числа, содержащие ```.``` или экспоненциальную ворму ```E```
- Ошибки чтения отдельных файлов не останавливают программу
- Если выходныя папка не существует - она создаётся автоматически
- Пустые выходные файлы не создаются

# Обработка ошибок

- Сообщает, если входной файл не найден
- Сообщает об ошибках чтения/записи
- Продолжает обработку остальных файлов при частичных ошибках
